Wow Travel - Time In Time Out System

A system designed to streamline the attendance monitoring system of the company.
It is designed to sync with the Company's Payroll app to simplify the process for both situations.


It is a facial recognition software.
It uses google sheets as it primary database for easy monitoring for the company.


It is to be run on a single computer per branch that allows any employee to log their Time In and Time Out without having to disturb admin.It syncs data to a google sheets that is accessible by the Company's Admins.
